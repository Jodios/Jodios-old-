Quick Sort

Description:

Quick sort is a divide and conquer sorting algorithm that partitions a list into smaller bits and sorts them recursively.

Complexity:

Quick sort's worst case scenario is O(n2), meaning that at its worst, it will make n2 comparisons; however, that only happens when you try and sort a list that is already sorted. On average, Quick Sort is O( nLog(n) )

Partitioning Schemes:

There are different ways to implement the partitioning of the list. This particular implementation is using the Lomuto partition scheme. Partitioning schemes do affect the performance of this algorithm, you can read more here.
